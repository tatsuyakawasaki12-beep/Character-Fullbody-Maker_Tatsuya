# Character Fullbody Maker — Tatsuya Edition Ver.2.0 Beta 19

## Beta19: Natural Body Hair System

### 主な更新
- ワキ毛を密度・面積・長さ・毛質・生える方向に分離
- Treasure Trailを太さ・濃さ・連続性・主分布への接続に分離
- メイン分布を開始位置・横方向・下方向・全体形状・境界・密度に分離
- 脚毛に長さと見え方を追加
- 安全表現でも選択した体毛を過度に弱めない技術資料向け補正
- 三面図は自動設定時に16:9の横長一列レイアウトを優先
- Beta18保存データからの移行処理

## 推奨
